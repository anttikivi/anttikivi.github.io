# Getting Support for the Website

If you’re experiencing issues with the website, please try the following steps before submitting an issue.

- Read [the documentation](#) of the website.
- Check [the FAQ](#) for a list of common questions and problems.
- Check if you can reproduce the problem [in the latest version of the website](https://github.com/anttikivi/anttikivi.github.io/releases/latest).
- Perform a [cursory search](https://github.com/issues?q=is%3Aissue+user%3Aanttikivi) to see if the problem has already been reported. If it has and the issue is still open, add a comment to the existing issue instead of opening a new one.

If your problem isn’t a bug or a similar issue but you need assistance with the website, please contact us at support@visiosto.fi.

If your problem is a bug, please [submit an issue](https://github.com/anttikivi/anttikivi.github.io/issues/new).
